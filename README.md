# PLC Projects

Industrial PLC programming projects using OpenPLC and STM32 Nucleo F411RE.

## Hardware
- STM32 Nucleo F411RE development board
- OpenPLC Editor (IEC 61131-3 ladder logic)
- Various sensors and actuators

## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | Button LED | Basic digital I/O - button controls LED |
| 02 | Latch Circuit | Industrial start/stop seal-in circuit |
| 03 | PLC Timers | Three core IEC 61131-3 timer instructions (TON, TOF, TP) |
| 04 | Counters | Event counting with CTU, CTD, CTUD instructions |
| 05 | Analog I/O | Temperature sensing and proportional control |
| 06 | Chiller Plant | Capstone - multi-stage HVAC control system |

## About
Self-directed PLC programming curriculum focused on building automation and HVAC applications.

## Skills Demonstrated
- IEC 61131-3 ladder logic programming
- Digital and analog I/O interfacing
- Industrial control patterns (latching, timing, counting)
- Hardware troubleshooting and debugging
- Technical documentation
